## Project Overview
This project demonstrates how to deploy a simple website on AWS EC2 using Nginx.

---

## Technologies Used
- AWS EC2
- Nginx
- Linux
- HTML

---

## Steps Performed

1. Launched an EC2 instance
2. Connected using SSH via Windows powershell
3. Installed Nginx
4. Deployed a custom HTML website
5. Accessed the site via public IP
